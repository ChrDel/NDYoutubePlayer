# NDYoutubePlayer



**NDYoutubeplayer** is a YouTube video player for iOS.

<img src="Screenshots/screenshoot_1.png" width="377" height="668">

## Requirements

- Runs on iOS 9.0 and later
- Requires Swift 4.0  and ARC.

## Installation

NDYouTubePlayer is available on CocoaPods.

CocoaPods:

```ruby
pod "NDYoutubePlayer"
```
